#### Information

#### Package Features
- login system
- console, bot control buttons


#### Package Example Usage
```javascript
const FGPanel = require("module-name");
const panel = new FGPanel({
  client: client /* discord client */,
  passwords: [...passwords]
});

const port = process.env.PORT || 3000;
panel.start(port);
```
